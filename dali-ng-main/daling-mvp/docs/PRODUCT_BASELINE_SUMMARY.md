# 搭里产品与合规基线摘要（执行版）

> 来源：你提供的三份基线文档（`README.md`、`baseline.md`、`搭灵 · 产品与合规基线包.md`）
> 用途：后续需求评审、实现、提测前的统一检查依据

## 1. 产品定位（硬边界）

搭里是“基于地理围栏 + 时间窗口”的活动信息展示系统，不是交易、支付、撮合、会员平台。

平台只做：
- 信息展示
- 信息排序
- 风险提示
- 认证与信任辅助

平台不做：
- 支付闭环
- 交易担保
- 履约承诺

## 2. 小程序内禁止项（强约束）

禁止能力：
- 支付、订单、退款、钱包、充值
- 明示价格、金额、会员费
- 会员/VIP/订阅/付费解锁

禁止文案（前端用户可见）：
- 支付、价格、订单、订阅、会员费、充值
- purchase/payment/order/checkout/membership/vip

## 3. 私域转化方式（允许项）

允许引导：
- 联系官方社群
- 添加官方微信获取活动编码
- 联系主理人确认细节

禁止引导：
- 任何价格暗示
- 任何付费路径暗示
- 小程序内交易闭环

## 4. 合规必备能力

- 发布活动需要实名认证
- 有用户举报入口
- 用户协议/隐私政策可访问
- 管理端具备快速下架能力

## 5. 开发执行规则（团队约定）

- 技术栈：uni-app + 微信云开发（维持现状）
- 不新增与付费相关字段/API 路径
- 需求不明确时先确认，不擅自补充高风险功能
- 每轮改动都执行本地回归清单并记录结果

## 6. 命名说明

你当前确认的小程序品牌名为“搭里”。  
历史文档中的“搭灵”视为同一产品早期命名，后续以“搭里”为准进行代码和文案统一。
